CHRISTIAN RAGONESE STUDENTE DEL CORSO FULL-STACK FS0225

-Il mio progetto si basa sulla realizzazione di un sito pensato per la vendita di biglietti per eventi sportivi (soffermandomi sul principale campionato Italiano Serie A). L'idea è quella di portare nello stesso sito gli eventi di tutte le squadre presenti nel campionato, fornendo una visione sulla classifica, eventi passati e prossimi.
-L'obiettivo è quello di portare una visione globale degli eventi calcistici, rendendone quanto più comodo l'acquisto, cercando di inserire quante più funzionalità possibili per rendere il sito tale.
Miro ad avere un sito chiaro ed intuitivo alla mano di tutti, cercando di rimanere nel semplice ma funzionale.
-L'intento è quello di utilizzare Java e PostgreSQL per il lato back-end, per quanto riguarda il front-end miro all'utilizzo del linguaggio Javascript con il supporto di Bootstrap e React per il completamento del mio progetto.
